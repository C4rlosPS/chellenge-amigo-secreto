# Projeto de Sorteio de Nomes

Este é um projeto simples de uma página web para sortear nomes. Os usuários podem adicionar nomes a uma lista e sortear um vencedor aleatório.

## Funcionalidades

- Adicionar nomes à lista
- Sortear um nome aleatoriamente da lista
- Limpar a lista de nomes 

## Tecnologias Utilizadas

- JavaScript
- HTML

## Como Usar

1. Digite o nome de um amigo no campo de entrada.
2. Clique no botão "Adicionar Amigo" para adicionar o nome à lista.
3. Após adicionar pelo menos dois nomes, clique no botão "Sortear Amigo" para sortear um nome da lista.

## Estrutura do Projeto

index.html: Arquivo principal contendo a estrutura HTML.
style.css: Arquivo de estilo CSS.
main.js: Arquivo JavaScript com as funcionalidades descritas.
